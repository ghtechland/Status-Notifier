Status Notifier

The busy light that syncs Microsoft Teams, Zoom, Slack & Discord — all at the same time.

Status Notifier is a USB desk busy light plus a free desktop app. When your status changes in Teams, Zoom, Slack, or Discord, the light changes color — instantly and automatically. No buttons, no manual switching. The people around you know you're on a call before they interrupt.

It's the only busy light that supports Discord, including server voice channel detection.
